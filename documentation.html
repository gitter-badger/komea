FAQ

  • What is Komea ?
    • Komea is an opensource framework to easily build indicator and metrics to monitor your development process and your build factory
  • How I can use it ?
    • Simply creates a maven project or gradle project and follow the quickstart. We will also provide some demo applications on our github repository. Stay informed!
  • What are the technologies used in Komea?
    • As a framework, we want to be the less intrusive in our customer software. We are mainly using Java, Groovy, OrientDB database and common libraries (guava, ...). Check our technological stack for more informations.
